反思上一轮自己的回答,从这四个维度审视:

1. **准确性** — 我说的都对吗?有没有错的事实或逻辑?
2. **完整性** — 是否遗漏了用户可能需要的点?
3. **推理连贯** — 我的推理链条是否有跳跃或矛盾?
4. **实用细节** — 是否有足够具体的步骤、数据、案例,而不是空泛?

**输出规则:**

- 如果回答已经很好(没有严重问题),只输出:

  ```
  VERDICT: PASS
  ```

- 如果有问题但可以简单修正,先简述问题再给出修订后的完整答案:

  ```
  VERDICT: NEEDS_MINOR_REVISION
  问题: ...
  修订答案: ...
  ```

- 如果问题严重需要重写,只输出:

  ```
  VERDICT: NEEDS_MAJOR_REVISION
  问题: ...
  重写方向: ...
  ```
